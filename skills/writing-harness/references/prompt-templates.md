# Prompt Templates

Use these when the user wants reusable prompts.

## 1. Rewrite for Authored Quality

```text
Rewrite this so it reads like it was written by a skilled human with:
- clear intent
- strong audience awareness
- concrete observations
- selective emphasis
- credible voice

Cut generic explanation.
Replace abstraction with specificity.
Strengthen the central claim.
Remove template energy.

Do not optimize for sounding casual or merely human-like.
Optimize for authored quality.
```

## 2. Evaluate a Draft

```text
Evaluate this draft using the following dimensions:
- clarity of purpose
- audience specificity
- point of view
- specificity and concreteness
- insight density
- structural focus
- voice credibility
- reader usefulness
- trustworthiness
- memorability

Then:
1. score each area from 1-5
2. identify the top failure modes
3. propose the highest-leverage rewrite moves
```

## 3. Build a Writing Spec

```text
Before drafting, create a writing spec with:
- target reader
- reader starting point
- core objective
- main claim
- what the piece must accomplish
- what it must avoid
- desired level of conviction
- 3-5 concrete observations to include
```

## 4. Editorial Pass

```text
Act as an editor, not a paraphraser.

Find and fix:
- generic abstraction
- over-explanation
- false balance
- stock phrasing
- weak examples
- uniform rhythm
- invisible speaker

Prioritize meaning, judgment, and usefulness over cosmetic variation.
```
