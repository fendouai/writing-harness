# Mode Selection

Use this before choosing how heavy the harness needs to be.

Not every task needs cognitive surgery.

## Decision Table

| Mode | Use When | Required Layers | Do Not Use When |
| :--- | :--- | :--- | :--- |
| `Light rewrite` | The draft mainly needs cleanup, compression, or anti-slop editing. | spec, thesis, light lint | The reader's frame must change. |
| `Editorial rewrite` | The draft needs stronger structure, sharper judgment, better specificity, or better rhythm. | spec, thesis, diagnosis, rubric | The task is high-resistance persuasion or strategic reframing. |
| `Cognitive rewrite` | The reader must leave with a different interpretation, priority, or decision frame. | baseline, AEH, modules, sensors | The task is simple proofreading. |
| `CTA surgery` | The task must transfer expert judgment under uncertainty, resistance, or complexity. | baseline, AEH, modules, CTA payload, sensors, rollback | The task is routine cleanup or short copy polish. |

## Fast Heuristic

Ask:

1. Does the reader only need cleaner wording?
   - Use `Light rewrite`.
2. Does the reader need a better argument?
   - Use `Editorial rewrite`.
3. Does the reader need a different judgment frame?
   - Use `Cognitive rewrite`.
4. Does the reader need to internalize an expert recognition or decision script?
   - Use `CTA surgery`.

## Hard Rule

If the task requires reader belief change and you choose `Light rewrite`, you are under-scoping the job.
