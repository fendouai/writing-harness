# Cognitive Sensors

Use these to check whether the draft is creating a real cognitive state transition.

## K Sensors

- distinction markers:
  - `rather than`
  - `instead`
  - `not`
  - `different from`
- observable action verbs
- concrete boundary conditions

## P Sensors

- if-then decision language
- sequence cues
- reusable tools:
  - blanks
  - checklists
  - templates

## A Sensors

- agency language rising
- passive or victim language falling
- resistance estimate falling across the draft
- identity or ownership language appearing near the end

## Rule

Do not rely on one sensor alone. Use them as a network.
